# Tmux layer

The tmux layer is active while holding the SPACE key.

The navigation keys below are according to their position on the EXTEND layer.

`‚¨Ö` : previous window  
`‚Æï` : next window  

`Ctrl` `‚¨Ö‚¨á‚Æï‚¨Ü` : move to pane  
`Shift` `‚¨Ö‚¨á‚Æï‚¨Ü` : resize current pane  

`PgUp` (L)  : move window to the left  
`PgDn` (Y)  : move window to the right  

`;`  : split vertically  
`/`  : split horizontally  

`K`  : command mode (`Prefix` + `:`)  

`B`  : reload config  

The modifiers are available on the left hand.
